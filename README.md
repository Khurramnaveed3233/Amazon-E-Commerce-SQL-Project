# SQL Portfolio Project: Amazon E-Commerce Database Analysis

##  Project Overview

This project focuses on analyzing an **Amazon-style e-commerce database** using **SQL** to extract meaningful business insights.  
The goal is to help stakeholders understand **sales performance, customer behavior, product profitability, and regional trends** for better decision-making.

The project simulates a real-world e-commerce scenario where raw transactional data is transformed into actionable insights through **advanced SQL queries**.

---

##  Business Problem

E-commerce platforms generate massive volumes of data, but without analysis, this data cannot support business growth.

The key business challenges addressed in this project are:

- Identifying **top-selling and low-performing products**
- Understanding **high-value customers**
- Measuring **profitability at product and category level**
- Analyzing **regional and monthly sales trends**
- Evaluating the **impact of discounts on sales**
- Supporting **marketing, pricing, and inventory decisions**

---

##  Dataset & Database Design

The database represents a simplified Amazon-like system and includes:

- Customers
- Products
- Orders
- Order Details
- Reviews & Ratings
- Wishlist Data
- Regional Information

A **relational database design** was used, ensuring:
- Data normalization
- Referential integrity using primary and foreign keys
- Efficient querying using joins

##  Key Insights Extracted

- A small number of products contribute to **majority of sales**
- High-spending customers represent strong **retention opportunities**
- Discounts increase sales volume but may reduce **profit margins**
- Certain regions outperform others, indicating **targeted marketing potential**
- Some low-selling products still have high profit margins
- Higher-priced products do not always result in lower demand
- Products with higher ratings generally show **better sales performance**

---

##  Business Impact & Recommendations

- Focus promotions on **high-wishlist but low-conversion products**
- Retain top customers using **loyalty and reward programs**
- Optimize pricing using **profit margin insights**
- Reduce inventory for consistently low-performing products
- Increase marketing spend in **high-revenue regions**
- Improve or remove low-rated products to enhance customer trust
